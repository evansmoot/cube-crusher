-ABOUT---------------------------------------------------------------------------
	Cube Crusher is a reimagination of the classic game Brick Breaker. The 
goal of the game is to bounce the red cube into all the rectangular bricks until 
no more remain on screen. If the cube passes the paddle and hits the bottom of
the screen it's game over. Good luck.

-CONTROLS------------------------------------------------------------------------
A - Play game from title screen
B - Launch cube
LEFT - Move paddle left
RIGHT - Move paddle right
SELECT - Reset game to title screen
